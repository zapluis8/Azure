Bienvenido a la API de Estudiantes

Utiliza las rutas de la API para interactuar con el sistema de estudiantes.

  • Obtener todos los estudiantes: /api/estudiantes
  • Agregar un estudiante: /api/estudiantes (POST)
  • Actualizar un estudiante: /api/estudiantes/<id> (PUT)
  • Eliminar un estudiante: /api/estudiantes/<id> (DELETE)
